# 001's Blog

这是我的个人博客，使用 Next.js + MDX + TypeScript + Tailwind CSS 构建。

## 技术栈

- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式
- **MDX** - 支持 Markdown 的文章
- **静态导出** - 快速部署

## 开始使用

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建静态文件
pnpm build
```

## 添加文章

在 `posts/` 目录下创建 `.mdx` 文件：

```mdx
---
title: "文章标题"
date: "2025-02-07"
excerpt: "文章摘要"
tags: ["标签1", "标签2"]
---

# 文章标题

正文内容...
```
